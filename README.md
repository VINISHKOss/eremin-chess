# Е-2 — Е-4 • Визитка шахматного педагога

Премиальная одностраничная визитка / лендинг для педагога дополнительного образования и руководителя шахматного клуба **Еремин Александр Алексеевич**.

## Особенности

- Полностью самодостаточный файл `index.html`
- Tailwind CSS через CDN (без сборки)
- Элегантный премиальный дизайн (тёмно-синий, кремовый, тёплое золото #d4af37)
- Стильная шахматная доска с анимацией хода **e2-e4** (ванильный JS)
- Полностью на русском языке
- Отзывчивый дизайн (идеально на мобильных)
- Плавная навигация + активное выделение разделов
- Красивая форма записи с модальным окном успеха
- Много белого пространства, премиальная типографика

## Структура разделов

1. Стикерная навигация
2. Hero (с интерактивной доской)
3. Обо мне
4. Шахматный клуб «Е-2 — Е-4»
5. Что дают занятия шахматами
6. Результаты и отзывы
7. Как записаться
8. Контакты + Footer

## Как запустить локально

**Просто дважды кликните на файл `index.html`** — сайт откроется в браузере.

Никакой сервер запускать не нужно — сайт полностью статический.

### Быстрый способ из терминала (macOS)

```bash
open e2e4-chess-visiting-card/index.html
```

**Рекомендуется** всегда открывать напрямую через файл.

## Развёртывание в интернет (полностью бесплатно)

Сайт статический — его можно выложить бесплатно.

### Лучший бесплатный вариант: GitHub + Netlify

1. Создай новый репозиторий на GitHub:
   - Name: `eremin-chess`
   - Public
   - **Не** создавай README, .gitignore и License

2. Запушь код (замени `USERNAME` на свой логин GitHub):

```bash
cd e2e4-chess-visiting-card
git remote add origin https://github.com/USERNAME/eremin-chess.git
git branch -M main
git push -u origin main
```

3. Задеплой на Netlify:
   - Зайди на https://app.netlify.com (войди через GitHub)
   - Нажми **Add new site → Import an existing project**
   - Выбери репозиторий `eremin-chess`
   - Build command оставь **пустым**
   - Publish directory оставь **пустым**
   - Нажми Deploy

После деплоя сайт будет доступен по адресу вида:
`https://eremin-chess.netlify.app`

2. Перетащите **всю папку** `e2e4-chess-visiting-card` (или хотя бы файлы index.html + папку images/) в окно браузера

## Интерактивная шахматная доска

Доска находится в Hero. Нажмите на неё или кнопку «Показать ход».

- Анимирует классический ход белых **e2 → e4**
- Поддерживает повторный запуск
- Двойной клик — сброс фигуры
- Работает на клавиатуре (Enter / Space)

## Технологии

- HTML5 + Tailwind CSS (CDN)
- Чистый vanilla JavaScript
- CSS Grid для шахматной доски
- Современные CSS-переходы и анимации
- Полная доступность (aria, keyboard, focus states)

## Советы по доработке

- Для реальной отправки формы подключите Netlify Forms или Formspree (просто добавьте атрибут `data-netlify="true"` на форму)
- Можно добавить настоящие фотографии с хорошим освещением в деловом стиле
- При необходимости можно расширить раздел «Достижения» реальными турнирными таблицами
